Prompt

Being a baseball fan for years and with a keen sense to data, I am recently looking into the sites provide MLB data visualization services, looking for opportunities to identify the design issue and rebuild a solution if possible. Some problems identified as being an user to sites like Fangraph and Statcast:

  1. Frustrated with the terms usage
  2. Too much information are displayed all at once.
  3. Not intuitive enough for uninitiated users.
  4. The table format of visualization is difficult to see the trend and pattern.

With these issues identified in mind, I was seeking for a solution which is intuitive, simple to use, and offer just enough information for a MLB player's performance status. I decided to build my own solution and hope this product will serve as an alternative solution for any person who is new to baseball and wants to step into the world of baseball data analysis.

* Note : The project purpose is only for personal, non-commercial usage, all the information displayed on the web app are proprietary content of MLB Advanced Media, L.P ("MLBAM").

User Story

The dashboard display a single player's career and seasonal statistics visualization based on users' query action. Note that I've disabled search function for player photo as what the screenshots display due to a potential violation for players' portrait right.

Product UI screenshots


Initial Search Interface

  • User can query a single player's information based on player name. ( Recommend using full name to search )
  • User can choose to query information of an currently active player or a resigned player.

Search Completion Interface

  • User can view player's basic demgraphical details such as age, height & weight, playing postion...etc.
  • User can view the career performance of the player and the career average performance per season, month, and week.
    ( Calculation is based on the approach provided by Baseball Reference )
  • User can hit the Glossary button to see detailed definition of the metrics displayed.
  • User can hit the Seasonal Data Dashboard button to display the performance trending of the queried player.
    ( Returned the latest 5 seasons of data, if available. )

Seasonal Data Dashboard Interface

  • User can view 3 types of chart for the player's performance trend.
  • User can filter the metrics to see different result in bar chart with career average per season as benchmark.
  • User can click the bars in bar chart to see different season's outing ratio in donut chart.
  • User can filter to see different metrics comparison display on the line chart.

    • ( *Note : The y-axis will adjust the labels based on the maximum filtering metrics. )
