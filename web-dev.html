Movie Trailer Website

Inspired by the project from Udacity which is developed using Python

→ VIEW PROJECT ←

User Story

  • Users can click the posters to see up to 15 movie trailers from Youtube.
  • Users can view the movies' introduction and IMDB's rating.
  • Users can decide what movies to display by filtering the genres or searching via key words.
  • Users can toggle the icon to view different version of trailers: The Official Trailer or the Honest Trailer ( Warning! Spoiler Alert ).

RGB Color Guessing Game

Project instructed by Colt Steele in his Web Development Bootcamp at Udemy

→ VIEW PROJECT ←

User Story

  • Users will guess the color corresponds to the RGB value showed on top by clicking the blocks.
  • Users can choose the game difficulty level by either the Easy or Hard mode.
  • Users can restart the game by clicking the NEW COLORS button.

Simple Todo List

Project instructed by Colt Steele in his Web Development Bootcamp at Udemy

→ VIEW PROJECT ←

User Story

  • Users can add new tasks by typing tasks name in the area which shows What to do Next ?.
  • Users can cross over the list which have been completed by clicking the tasks.
  • Users can delete the task by hover over the tasks and click the red trash icon on the left.
  • Users can show or hide the task input section by toggling the circle icon on the top right corner.
